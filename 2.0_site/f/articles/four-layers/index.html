The Four Layers of Business Math

How symbols compile across abstraction layers without losing meaning.

All symbols in Intent Tensor Theory compile across four distinct layers. Each layer has its own rules. Violating layer boundaries produces nonsense.

Layer 0

GlyphMathMeaning, pre-coordinate. Pure semantic content before measurement. Never compute here.
Layer 1

TheoreticalField theory, functional analysis. Continuous operations on infinite-dimensional spaces.
Layer 2

ClassicalVector and tensor calculus. Finite-dimensional approximations suitable for engineering.
Layer 3

StandardScalars, derivatives, computation. Numbers you can put in a spreadsheet. Never philosophize here.
The Rule

Never compute in GlyphMath. Never philosophize in Standard Math.

When someone tries to calculate the meaning of Φ (Layer 0 operation in Layer 3), they produce garbage. When someone tries to find deep meaning in a KPI (Layer 3 content treated as Layer 0), they produce bullshit. The layers exist for a reason.
