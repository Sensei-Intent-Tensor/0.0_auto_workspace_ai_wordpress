### 1.1.2 Manual Steps Required (Browser — 5 min)

**WordPress.com Dashboard:**
1. wp-admin → Upgrades → Domains → auto-workspace-ai.com
2. Disable domain lock (transfer lock off)
3. Request EPP/Auth code (sent to registrant email)
4. Disable WHOIS privacy temporarily if enabled

**Cloudflare Dashboard:**
1. dash.cloudflare.com → Domain Registration → Transfer Domain
2. Enter: auto-workspace-ai.com
3. Paste EPP code
4. Confirm WHOIS contact info
5. Pay transfer fee (1 year extension, at-cost ~$9-10)
6. Approve transfer email from current registrar

Transfer duration: 5–7 days (ICANN mandatory hold)
